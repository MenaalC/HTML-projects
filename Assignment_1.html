Welcome to HTML

Explanation on all the basic structures of HTML: 1. "!DOCTYPE html" tells us that this is a a HTML5 file 2. "html lang="en"" tells us that the language of the browser will be English 3. "head" tells us that this part has all the information about the page 4. "meta charset="UTF-8"" tells us that we can display any character 5. "meta name="viewport" content="width=device-width, initial-scale=1.0" makes the page responsive, the page matches the width of the device and it is to scale, there is no zoom" 6. "title" tells us the title on the browser tab 7. "head" tells us information about the page 8. "body" tells us what will be displayed on the page 9. "p" tells us what the regular text will have

To run a HTML program, you first need to create a file using a text editor like VS and save it under ".html" once it is made and saved use the necessary basic structure of HTML to write what you want your page to say once completed open your page in the web browser.
